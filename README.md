Site one-page que apresenta o modelo de operação (Engenharia · Psicanálise · Estética High-End), os planos de serviço e o diferencial de **Portal do Cliente** próprio.

## 🔗 Live
https://websrchz.github.io/SrChZ-Technologies/

## ✨ Destaques
- Design **preto / branco / dourado**, totalmente responsivo (mobile-first).
- Acessível: navegação por teclado, `prefers-reduced-motion`, contraste alto, skip-link, ARIA.
- SEO: meta tags, **Open Graph / Twitter Card**, **JSON-LD** (`ProfessionalService`), `sitemap.xml`, `robots.txt`.
- PWA-ready: `manifest.webmanifest`, favicon SVG, `theme-color`.
- Sem dependências de build — HTML, CSS e um JS leve (IntersectionObserver para reveal e scroll-spy).

## 🗂 Estrutura
```
index.html              página principal (one-page)
404.html                página de erro
css/styles.css          estilos + responsividade + animações
js/main.js              nav, menu mobile, reveal, back-to-top, scroll-spy
assets/favicon.svg      ícone (monograma RC)
assets/og.png           imagem social (gerada de assets/og-image.html)
manifest.webmanifest    PWA
robots.txt · sitemap.xml
```

## 🚀 Rodar localmente

```bash
python -m http.server 8080

```
